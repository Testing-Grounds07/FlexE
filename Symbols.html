Typical style tags and placeholder symbols you can use to format the look of the final document include:

Style Tags

Headers

$...$$$$$$

Formats the content with a header style between 1 and 6.
 $$$$ Title

Title
List Items

@...@@@@@@

Formats the content into a list and specifies the indention up to level six.

    Comments

    //

    Excludes the content from the document. Useful for spreadsheet comments and directions.
    		 // This is a comment

    Does not appear in the final document.
    Bold

    [b]text[/b]

    Formats the wrapped text with a bold style.
    		 [b]Sample Text[/b]

    Sample Text
    Italic

    [i]text[/i]

    Formats the wrapped text with an italic style.
    		 [i]Sample Text[/i]

    Sample Text
    Underline

    [u]text[/u]

    Formats the wrapped text with an underlined style.
    		 [u]Sample Text[/u]

    Sample Text

    Placeholder Symbols

    Document Title

    !x!


    Adds the title of the document anywhere the symbol appears (lowercase only).
    		 !x!

    document title
    Product Matrix

    !ProductMatrix!

    Adds a full product matrix anywhere the symbol appears.
    		 !ProductMatrix!
    Product 1
    Product 2
    Product List

    !ProductList!

    Adds a full product list anywhere the symbol appears.
