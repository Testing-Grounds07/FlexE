Typical style tags and placeholder symbols you can use to format the look of the final document include:

Style Tags

  • Headers | $...$$$$$$ - Formats the content with a header style between 1 and 6. For example: "$$ Title" adds a header 2 style.
  • List Items | @...@@@@@@ - Formats the content into a list and specifies the indention up to level six. For example: "@@@ Item" adds a list item with three indentions.
  • Comments | // - Excludes the content from the document. Useful for spreadsheet comments and directions. For example: "// This is a comment" will not appear in the document.

Placeholder Symbols

  • Document Title | !x! - Adds the title of the document anywhere the symbol appears (lowercase only).
  • Product Matrix | !ProductMatrix! - Adds a full product matrix anywhere the symbol appears.
  • Product List | !ProductList! - Adds a full product list anywhere the symbol appears.
